# 전자와 결합의 세계
### 탄수화물에서 구리 전선까지 — 인터랙티브 화학 학습 웹앱

## 실행 방법

```bash
cd electron-world-app
npm install
npm run dev
```

브라우저에서 http://localhost:5173 을 열면 됩니다.

## 기술 스택

- React 19 + Vite 8
- Tailwind CSS v4 (@tailwindcss/vite)
- 순수 CSS / SVG / Canvas 애니메이션 (외부 API 없음, 한국어 전용)

## 주요 기능

| 기능 | 설명 |
|------|------|
| 비유 모드 / 정확 모드 | 상단 토글로 쉬운 비유 ↔ 과학적 설명 전환 |
| 진행률 추적 | 완료한 모듈이 localStorage에 자동 저장됨 |
| 미니 퀴즈 | 각 모듈 끝 O/X 퀴즈 + 친절한 피드백 |
| 전체 퀴즈 | 12문항 O/X, 점수 및 레벨 표시 |
| 영어 요약 | 요약 카드 화면에서 영어 전체 요약 확인 |
| 반응형 UI | 모바일/태블릿/데스크톱 모두 지원 |

## 모듈 구성

1. 홈 — 파티클 배경 애니메이션
2. 학습 지도 — 모듈 카드 + 개념 흐름도
3. 모듈 1: 원자와 전자 — 전자 추가/제거 인터랙션
4. 모듈 2: 수소 H, H⁺, H⁻ — 변환 시뮬레이터
5. 모듈 3: 산성과 -COOH — pH 슬라이더
6. 모듈 4: 비타민 C와 항산화 — 전자 전달 애니메이션
7. 모듈 5: 탄수화물·지방·아미노산 — 에너지 비교 차트 + 에스터 결합
8. 모듈 6: 광합성과 탄소 포집 — 단계별 전자 들뜸 시뮬레이션
9. 모듈 7: 오존과 활성산소 — O₂ vs O₃ 비교
10. 모듈 8: 금속 결합 — Canvas 전자 흐름 애니메이션
11. 모듈 9: 녹·산화·보호막 — 철/구리 부식 진행 시뮬레이션
12. 퀴즈 — 12문항 전체 테스트
13. 요약 카드 — 핵심 10문장 + 영어 요약
